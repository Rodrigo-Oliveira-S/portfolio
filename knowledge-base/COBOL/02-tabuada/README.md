## Tabuada em COBOL

Programa que exibe a tabuada de um número utilizando estruturas de repetição.

## Objetivo

Praticar o uso de loops em COBOL, compreendendo o funcionamento do PERFORM VARYING
 para controle de repetições e sua aplicação em problemas matemáticos simples.

## Conceitos Aprendidos

- Uso do comando PERFORM VARYING para criar loops
- Operações aritméticas dentro de repetições com COMPUTE
- Exibição de dados com DISPLAY em múltiplas linhas
- Controle de fluxo em estruturas de repetição

## Como Executar

cobc -x -o tabuada tabuada.cbl

./tabuada
