## Contador em COBOL

Programa que conta de 1 até um número definido utilizando loops.

## Objetivo

Praticar o uso de diferentes estruturas de repetição em COBOL, compreendendo o funcionamento do PERFORM UNTIL e suas variações.

## Conceitos Aprendidos

- Uso do comando PERFORM UNTIL para loops com condição de parada
- Incremento de variáveis com ADD
- Entrada de dados para definir limites
- Controle de fluxo em estruturas de repetição
- Exibição sequencial de valores

## Como Executar

cobc -x -o contador contador.cbl

./contador
